# 基本注意事项
## 穿衣部分
徒步这项运动，需要在户外进行长时间的步行，因此会面临出汗的情况比较多。当你出汗没有及时进行排汗的话，很容易会使得徒步的过程中感到不舒服，甚至生病。同时，可能在徒步的过程中会出现海拔变化，从而带来温度的变化。因此我们需要及时进行衣物的穿着和脱下。


穿衣部分我们讲究一个“洋葱法”：

- 内层（排汗衫）：内层部分需要及时进行排汗，所以内层部分需要透气性强、弹性好。最好选择比较专业的排汗衫，并且拒绝纯棉的衣服，在温度低的环境下，汗液在纯棉的衣服下不容易排出。
- 中层（保暖）：保暖层需要羽绒服，或者抓绒的衣物。这一层主要的功能就是保暖，当然也会需要一定的透气性。
- 外层（防风）：最外层可有可无，主要是面对一定的大风天气或者潜在性的下雨天，从而需要最外层的防风层，这一层最好是专业的冲锋衣。
- 袜子：徒步袜或者羊毛袜，耐磨、并且防止冻伤。
- 鞋子：看徒步的难度，来选择徒步鞋的种类。最好选择带有一定防水能力的徒步鞋。
- 手套：预防温度的下降，需要一定防风功能的手套，这样在户外拿取一些工具的时候，手不会冷。
- 帽子：温度较低的情况下，毛线帽是很必要的。但如果是在温度较高的情况下，就可以戴鸭舌帽或者徒步帽。温度如果只是在 5 度（北方）左右，那么就没有必要戴毛线帽，早上冷的话，可以多戴一个，上山之后换掉。并且毛线帽也是最好要选择有弹性的，适度紧贴头部的，松松垮垮的帽子在出汗之后，会很难受。

## 补剂部分
补剂讲究轻量、每一次的量讲求恰当，这一点和自行车运动都是类似的。

- 由于运动的时间比较长，重点需要关注的就是补水，并且为了更高效的补水效果，可以再带一瓶电解质水，在出汗多的时候及时补充。
- 能量补剂也是一样的道理，可以带能量胶，或者一些随手可以方便吃的食物，如果徒步的时间较长，那么就可以准备野炊的物品。但是要注意的是，糖分补充的过多的话，胃也不好消化，所以避免频繁的高糖补充。


## 装备部分

- 灯具：防止天黑的情况，因此需要带一个灯具。
- 充电宝：及时为通讯设备供电。
- 垃圾袋：收拾自己的垃圾，不要留在户外。
- 头巾/手巾：擦汗。
- 卫生湿纸巾：多种用途。
- 创可贴：医用需求。
- 一些药品：胃药、退烧药、消炎药、碘伏等等。


## 其他事项

- 尽量在白天完成徒步，天黑之后，会存在很多的危险，并且在天黑时景区一般不让游客进行徒步活动。
- 防晒部分需要注意，可以带墨镜，在脸上涂防晒霜。
- 通讯设备需要充满电，并且提前下载好离线地图，防止在户外没有信号。
- 最好要提前设定好路线，查看每一个路线的那一程度。
